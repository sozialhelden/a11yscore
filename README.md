# a11yscore

## Requirements

* [bun](https://bun.sh/) v1.2 or greater

## Getting started

```bash
# install dependencies
bun install

# create a .env file and edit it accordingly
cp .env.example .env

# start the development server
bun run dev
```

## Useful commands

```bash
# format the code
bun run format

# lint the code
bun run lint

# run tests, make sure the development server is running
bun run test
```
